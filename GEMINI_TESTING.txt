GETTING STARTED WITH GEMINI (3 MINUTES):
========================================

1. Get free API key:
   - Go to: https://aistudio.google.com/app/apikey
   - Click "Create API key"
   - Copy the generated key

2. Add to .env:
   AI_PROVIDER=gemini
   GEMINI_API_KEY=your-key-here
   GEMINI_SPEC_MODEL=gemini-1.5-flash
   GEMINI_PROC_MODEL=gemini-1.5-flash

3. Run conversion:
   export AI_PROVIDER=gemini
   python anything_to_spec.py \
     --programs_directory ./tests/cobol/test1/ \
     --spec_path ./outputs/spec1.yml

4. Generate procedures:
   python spec_to_xpz.py \
     --spec_path ./outputs/spec1.yml \
     --output ./outputs/test1

5. Import into GeneXus:
   Open GeneXus and import: outputs/test1/import_file.xml

CONFIGURATION OPTIONS:
====================

GEMINI_API_KEY:
  Required: YES
  Example: AIzaSyDummyKeyForTesting...
  Get from: https://aistudio.google.com/app/apikey

GEMINI_SPEC_MODEL:
  Default: gemini-1.5-flash
  Other options: gemini-1.5-pro, gemini-2.0-flash
  Used for: Code analysis → Specification

GEMINI_PROC_MODEL:
  Default: gemini-1.5-flash
  Other options: gemini-1.5-pro, gemini-2.0-flash
  Used for: Specification → GeneXus Procedure

TROUBLESHOOTING:
================

Error: "GEMINI_API_KEY not set"
  → Solution: Add GEMINI_API_KEY=your-key to .env

Error: "API error"
  → Reason: Invalid key or API down
  → Solution: Check key at https://aistudio.google.com/app/apikey

Error: "Rate limit exceeded"
  → Reason: More than 15 requests/min or 1M tokens/day
  → Solution: Wait or upgrade to paid tier
