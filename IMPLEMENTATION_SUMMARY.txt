USAGE EXAMPLES
================================================================================

OPTION 1: OpenAI (Your Paid Plan)
  export AI_PROVIDER=openai
  export OPENAI_API_KEY=sk-your-api-key
  python anything_to_spec.py --programs_directory ./tests/cobol/test1/ \
    --spec_path ./outputs/spec1.yml

OPTION 2: GitHub Copilot
  export AI_PROVIDER=copilot
  python anything_to_spec.py --programs_directory ./tests/cobol/test1/ \
    --spec_path ./outputs/spec1.yml

OPTION 3: Google Gemini (FREE - No setup beyond API key)
  export AI_PROVIDER=gemini
  export GEMINI_API_KEY=your-free-api-key
  python anything_to_spec.py --programs_directory ./tests/cobol/test1/ \
    --spec_path ./outputs/spec1.yml

OPTION 4: Ollama (LOCAL - Completely Free & Private)
  export AI_PROVIDER=ollama
  ollama serve  # Start in background
  python anything_to_spec.py --programs_directory ./tests/cobol/test1/ \
    --spec_path ./outputs/spec1.yml

OPTION 5: GXEAI (Legacy)
  export AI_PROVIDER=gxeai
  export SAIA_PROJECT_APITOKEN=your-token
  python anything_to_spec.py --programs_directory ./tests/cobol/test1/ \
    --spec_path ./outputs/spec1.yml

OPTION 6: Command Line Override
  python anything_to_spec.py \
    --programs_directory ./tests/cobol/test1/ \
    --spec_path ./outputs/spec1.yml \
    --provider openai

NEXT STEPS
================================================================================

For Production Use:

1. Choose your provider based on needs:
   - OpenAI: Best quality, fast, but requires API key and costs
   - Copilot: Very fast, integrated with GitHub Copilot subscription
   - Gemini: Free tier with generous limits, no credit card
   - Ollama: Completely free, private, local (need 16GB+ RAM)
   - GXEAI: Legacy option if you have existing token

2. Install dependencies:
   pip install -r requirements.txt

3. Set up your chosen provider:
   - OpenAI: Add OPENAI_API_KEY to .env
   - Gemini: Add GEMINI_API_KEY to .env (get free at https://aistudio.google.com/app/apikey)
   - Ollama: Install from https://ollama.ai and run "ollama serve"
   - Copilot: Ensure local CLI is installed
   - GXEAI: Add SAIA_PROJECT_APITOKEN to .env

4. Set AI_PROVIDER in .env or via command line:
   export AI_PROVIDER=gemini  # or openai, copilot, ollama, gxeai

5. Run end-to-end test:
   python anything_to_spec.py --programs_directory ./tests/cobol/test1/ \
     --spec_path ./outputs/spec1.yml
